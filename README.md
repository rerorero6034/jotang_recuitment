# 部署方式
下载文件后，分为两个文件夹，blog中为题目一，bilibili中为题目二。其中题目一的网页入口文件为blog.html，题目二的网页入口文件为source文件夹中的bilibili.html。在vs中打开对应文件，在浏览器中预览即可看到网页效果
题目一有百叶窗图片，侧边栏功能
题目二实现了题目要求里的绝大部分功能
文件均支持响应式布局，不同的显示缩放率均可正常显示，不过我是笔记本，显示缩放率为125%，为获得最好的观看体验，建议将缩放率调至125%
